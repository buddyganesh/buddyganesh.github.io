# buddyganesh.com

Personal author website for Buddy Ganesh — home of The Healing Series, a six-volume collection of guided reflection journals.

**Live site:** [buddyganesh.com](https://buddyganesh.com)

## Files
- `index.html` — Main landing page (The Healing Series)
- `404.html` — Custom not-found page
- `CNAME` — Custom domain configuration for GitHub Pages

## To update Amazon links
Open `index.html` and search for `href="#"` — replace each `#` with the appropriate Amazon product URL for each volume.

## Hosting
Hosted on GitHub Pages with a custom domain via Namecheap/GoDaddy DNS.
